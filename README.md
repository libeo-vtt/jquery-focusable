# jQuery :focusable Selector Plugin

## Usage

1. Include jQuery:

	```html
	<script src="path/to/jquery.min.js"></script>
	```

2. Include plugin's code:

	```html
	<script src="path/to/jquery.focusable.min.js"></script>
	```

3. Use the selector:

	```javascript
	$(':focusable');
	$(document).find(':focusable');
	```
